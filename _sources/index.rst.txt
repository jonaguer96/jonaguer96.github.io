Welcome to Learning Sphinx documentation for work!

Welcome to the complete beginner's guide to Sphinx! 
If you're looking for compregensive guide to the Sphinx documentation markdown language, then you've come to the right place



Who is this for?


You don't need to know anything about programming to begin this course, this is completely beginner-friendly! Even if you have some
coding experience, you'll be sure to find this guide useful. This guide is for anyone creating documentation help document for their
respective capability.



What is restructuredText?

reStructuredText is an easy-to-read, what-you-see-is-what-you-get plaintext markup syntax and parser system. It is useful for in-line program documentation
(such as Python docstrings), for quickly creating simple web pages, and for standalone documents. All throughout the course, we will be covering what one needs
to install to write documentation with restructured text
formatting with Sphinx via VS Code.
